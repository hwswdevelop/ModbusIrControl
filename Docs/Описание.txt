Интерфейсный модуль предназначен для установки в коробку выключателя.
Модуль входжит в состав системы умный дом, опршивает клавиши включателя, 
инфракрасный порт и пердает состояние выключателя на шину RS485.
Питание модуля производится с линии 12В

Порты:
1. Питание 12В (клеммники)
2. RS585 (клеммники)
3. Входы выключателя (клеммники)
4. Три независимых выхода для подключения подсветки клавиш (разъем). Позволяет отображать статус (включено - выключено)
5. Вход ИК порта. Может импользоваться для дистанционного управления стандартными ИК пультами управления. Поддерживается NEC длинна посылки от 32х до 96 бит. 
6. Выход ИК порта. Может использоваться для передачи команд. Например для включения/отключения кондиционера по времени или через интернет.
